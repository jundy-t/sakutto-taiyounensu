# サクッと耐用年数 — SEOキーワード調査

Phase 1 title/meta 最適化のためのキーワード調査ログ置き場。

## 保存してほしいファイル

| ファイル名 | 内容 | 取得元 |
|-----------|------|--------|
| `rakkokeyword_relatedKeywords_耐用年数_YYYY-MM-DD_*.csv` | 「耐用年数」のラッコサジェスト（月間検索数・SEO難易度付き） | ラッコキーワード有料プラン |
| `gsc-queries-sakutto-taiyounensu-YYYYMMDD.csv` | 過去28日〜3ヶ月の実表示クエリ一覧（取れれば） | Google Search Console |

## ラッコキーワード操作手順

1. https://related-keywords.com/ にアクセス・ログイン
2. 検索窓に `耐用年数` と入力 → 「無料で調査」
3. 「月間検索数/SEO難易度（最新）を取得」ボタンをクリック → 数値が列に追加される
4. マッチタイプ: 部分一致（デフォルト）のまま、フィルタ解除で5,000件全体を取得
5. 画面右上「データ出力」→ **CSV** → ダウンロード → このフォルダに移動

## GSC操作手順（取れれば任意）

1. https://search.google.com/search-console にアクセス
2. プロパティ `sakutto-taiyounensu.haraochi.jp` を選択
3. 左メニュー「検索パフォーマンス」→「クエリ」タブ
4. 期間: 過去3ヶ月（28日でも可）
5. 画面右上「エクスポート」→ CSV → このフォルダに保存

**注意**: サクッと耐用年数も新規サイトならクエリがゼロの可能性あり。その場合は ラッコ CSV だけで進めます。

## データ揃ったら

「取得完了」と Claude に伝えてください。Claude 側で読み込み → title / meta / OGP / JSON-LD の設計案を提示します。

## 想定 sweet spot 範囲

サクッと経費判定の経験を元に：
- 月間検索数 500〜5,000 の帯で SEO難易度 20〜33 のロングテール語を優先
- 競合が freee/マネーフォワード/税理士メディアで埋まっている高難度語は避ける
- 「耐用年数 パソコン」「耐用年数 車」「中古資産 耐用年数」「減価償却 耐用年数」等の複合語が狙い目の想定
